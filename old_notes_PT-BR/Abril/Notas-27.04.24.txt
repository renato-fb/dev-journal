== Anotaçãoes para reenforçar o conhecimento ===
React é uma biblioteca que tem como o propósito principal aplicar reatividade as aplicações web, criando uma experiência dinâmica melhor usada em aplicações web de complexidade mediana-grande. 

Piece of Data = Piece of State.

Um grande positivo de usar React é que o mesmo possui um jeito correto de estruturar seu código, diminuindo as chances do famoso código espaguete, outro grande positivo é que react dá disponibiliza uma forma mais consistente de desenvolver aplicações entre equipes e desenvolvedores.
